Online, offline, and anything in between

Use topics to organize the discussion

  • Zulip topics create a separate space for each discussion. Instructors don’t need to worry about missing a question, and can mark a topic “resolved” when done.
  • With topics, answers are easy to find, so each question is only asked once.
  • Refer to relevant points by linking to a message or a topic.
  • Keep it orderly by moving messages and topics when the discussion takes a turn.

Powerful formatting

  • Structure your points with bulleted and numbered lists.
  • Teaching technical topics? Type LaTeX directly into your Zulip message, and see it beautifully rendered.
  • Discuss code with ease using code blocks, with syntax highlighting for over 250 languages, and integrated code playgrounds.

I used Piazza & Zulip. Piazza lets students ask questions anonymously. Zulip better all-around for chatting, announcements, group work (has built-in LaTeX, as well as threading). Zulip basically replaced email for my class. Easy to pick up if you've used Slack & email before.
